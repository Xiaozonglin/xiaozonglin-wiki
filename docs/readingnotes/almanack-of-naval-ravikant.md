# 读《纳瓦尔宝典》

## 第一部分：财富

这一部分读得不太明白，感觉讲的道理都是自己能够听得懂的，但知易行难。自己目前所选的专业到底是不是自己想要的专业？自己想要从事什么样的职业？这些现在还没有一个很好的答案。「社会有需求但无从获得的东西」？我似乎没有发现这种需求的眼睛。

- 赚钱和工作的努力程度没什么必然联系。与埋头苦干相比，更重要的是理解和思考。当然，努力非常重要，不能吝啬自己的努力，但必须选择正确的方式。
- 如果还不知道自己应该做什么，那么你先要弄清楚这个问题。
- 财富是在睡觉时也能带来收入的企业和资产。
- 要想在社会上赚到钱，就要为社会提供其有需求但无从获得的东西。
- 专长无法被教授，但可以被学习。要积累和发展专长，就要发挥自己的天赋，研究自己真正好奇的东西，追寻自己的热情所在，而不是选择一个当下的热门专业，然后在毕业后进入投资者声称的热门专业。
- 致富最重要的技能是成为终身学习者。用头脑赚钱，而不是用时间赚钱。
- 出发点并不重要，行为本身才重要。
- 99%的努力终将白费，努力找到不会被浪费的1%。这1%对你是有意义的，值得你用余生去追求。一旦找到，你就要心无旁骛，全力以赴。
- 获得财富需要杠杆，杠杆可以来自劳动力、资本、代码或媒体。
- 如果一件事物一开始让你兴致盎然，后来又让你觉得索然无味，那么它只是暂时分散了你的注意力，并不是你心智上真正的好奇所在。请继续寻找。
- 对一件事情的欲望越小，顾虑就越少，执念就越少，反而越会顺其自然，遵循内心。
- 投入和产出之间存在不匹配性，特别是对那些利用了杠杆效应的劳动者来说。
- 学会销售，学会构建。同时掌握这两种技能的人会势不可挡。
- 随着层级不断向上，杠杆效应愈加明显，责任愈加重大，需要的专长也越来越多。
